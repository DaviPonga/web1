Atividade 1 — Flexbox Froggy

Joguei todos os níveis. Abaixo, dois níveis acima do 10 com o CSS usado e o efeito de cada propriedade.

Nível 13

#pond {
  display: flex;
  flex-direction: row-reverse;
  align-items: flex-end;
  justify-content: center;
}
  • flex-direction: row-reverse — inverte a ordem dos itens (o primeiro vai pra direita).
  • align-items: flex-end — alinha todos no final do eixo vertical (fundo do tabuleiro).
  • justify-content: center — agrupa os itens no centro do eixo horizontal.

Nível 17

#pond {
  display: flex;
}

.frog-yellow {
  align-self: flex-end;
  order: 1;
}
  • align-self: flex-end — alinha só esse item no final do eixo vertical, sem afetar os outros.
  • order: 1 — muda a posição desse item na fila (padrão é 0); quem tem valor maior aparece depois.
