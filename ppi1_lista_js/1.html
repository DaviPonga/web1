O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, interpretada e voltada para objetos, essencial para tornar páginas web interativas e dinâmicas.

O que é JavaScript?

A motivação para a criação do JavaScript.

A motivação principal foi adicionar dinamismo e interatividade às páginas web, que na época eram estáticas.

A diferença entre execução no navegador e no servidor.

A execução no navegador (Client-Side) ocorre no dispositivo do usuário, focando na interface e interatividade.

O conceito de sandbox no navegador.

É o vidro blindado do seu computador. Ele deixa você ver a internet, mas não deixa a internet tocar em você.

Objetos Globais como document, console e alert.
