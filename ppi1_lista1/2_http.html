Protocolo HTTP

O que é HTTP?

O HTTP (HyperText Transfer Protocol) é basicamente a forma que os computadores usam para se comunicar na internet. É ele que permite que o navegador (como Chrome, Edge, etc) peça informações para um servidor.

Por exemplo: quando você digita um site, o seu navegador manda uma requisição pedindo aquela página. O servidor recebe isso e manda de volta a resposta com o conteúdo.

Como funciona?

O HTTP funciona com um sistema simples: pedido e resposta.

O navegador (cliente) faz um pedido → o servidor recebe → e responde com o que foi pedido.

Exemplo de Requisição HTTP

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla/5.0
Accept: text/html

Explicação:

  • GET → tipo de requisição (aqui ele está pedindo dados)
  • /index.html → arquivo que está sendo solicitado
  • HTTP/1.1 → versão do protocolo
  • Host → qual site está sendo acessado
  • User-Agent → qual navegador está fazendo o pedido
  • Accept → tipo de conteúdo que o navegador aceita

Exemplo de Response HTTP

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1024

<html>
    <body>Olá mundo</body>
</html>

Explicação:

  • HTTP/1.1 → versão usada
  • 200 OK → significa que deu tudo certo
  • Content-Type → tipo do conteúdo enviado
  • Content-Length → tamanho da resposta
  • Body → conteúdo que foi enviado (a página em si)

Códigos de Status HTTP

  • 200 → tudo certo
  • 404 → página não encontrada
  • 500 → erro no servidor

Conclusão

O HTTP é essencial para a internet funcionar. É através dele que acontece toda a troca de informações entre o navegador e os servidores. Sem ele, basicamente nenhum site abriria.
