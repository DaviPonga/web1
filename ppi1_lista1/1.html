Diferença entre Internet e Web

A Internet é a infraestrutura global que conecta computadores no mundo todo. Já a Web é um sistema de páginas e informações acessadas por navegadores. A Internet é como a estrada, enquanto a Web é o conteúdo que circula nela.

História da Internet e da Web

Tim Berners-Lee criou a Web em 1989. Ele trabalhava no CERN e queria facilitar o compartilhamento de informações.

Principais marcos

  • 1969 - Criação da ARPANET
  • 1983 - Surgimento do protocolo TCP/IP
  • 1989 - Criação da Web
  • 1991 - Primeiro site no ar

Ordem dos acontecimentos

  1. Internet criada
  2. Protocolos definidos
  3. Web criada
  4. Popularização

Definições

Internet
Rede global de computadores. Ela funciona por meio de protocolos como o TCP/IP e inclui toda a infraestrutura física, como cabos, satélites, roteadores e servidores. É a base que permite serviços como e-mail ou mandar mensagens para pessoas que estão conectadas à internet, jogos multiplayer/online e a própria Web existirem.
Web
Sistema de páginas acessadas via navegador. Consiste em páginas, sites e conteúdos acessados por navegadores (como Chrome, Firefox, Edge entre outros) dando também para acessar ate mesmos jogos de navegadores. Usa tecnologias como HTTP(HyperText Transfer Protocol), HTML(HyperText Markup Language) e URLs(Uniform Resource Locators) para organizar e exibir informações em forma de sites interligados por links.
