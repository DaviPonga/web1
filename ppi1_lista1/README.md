Projeto Web - HTML e CSS

Esse projeto foi feito como atividade da matéria, com o objetivo de praticar HTML e CSS.

---

--O que tem no projeto:

O projeto tem algumas páginas:

- 1.html → fala sobre a história da Internet e da Web  
- 2_http.html → explica o que é o protocolo HTTP  
- 3_tabela_uf.html → mostra uma tabela com estados do Brasil  
- 4_contato.html → formulário de contato  
- 4_curriculo.html → página de currículo  

---

CSS:

Foi usado um arquivo externo chamado:

- style.css  

Nele eu coloquei:
- cor de fundo da página  
- cor dos textos  
- fonte padrão (Arial)  
- tabela com bordas e espaçamento  
- inputs arredondados  
- botão com efeito hover  
- alguns estilos pra deixar mais bonito

Objetivo:

O objetivo foi aprender:

- como montar páginas em HTML
